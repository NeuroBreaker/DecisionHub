# 🚀 DecisionHub

---

## ⚡ Быстрый запуск через Docker

```bash
docker-compose up -d --build
```

> Веб-страница будет доступна по адресу: **http://localhost/**


## 🛠 Технологический стек

| Слой | Технологии |
|------|-----------|
| **Frontend** | React, TypeScript, Tailwind CSS, Shadcn/ui, Zustand, Vite |
| **Backend** | Python, FastAPI, SQLAlchemy, Uvicorn, UV (менеджер пакетов) |
| **База данных** | PostgreSQL |
| **Анализаторы** | Модули статического анализа кода (LOC, цикломатическая сложность, поиск секретов через регулярные выражения), анализаторы PDF/DOCX структуры и парсеры видео-ссылок ВК/YouTube |

---

## 📐 Архитектура проекта

Приложение построено по **клиент-серверной архитектуре**:

- **Frontend (SPA)** — взаимодействует с бэкендом через REST API. Адаптирован под три роли: Участник, Член жюри и Организатор.
- **Backend (FastAPI)** — обрабатывает бизнес-логику, авторизацию и запускает модули анализа артефактов в оперативной памяти без засорения диска временными файлами.
- **Database (PostgreSQL)** — хранит данные пользователей, команд, загруженных результатов автопроверок и экспертных оценок.

---

## 🔌 Основные API эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| `POST` | `/api/auth/register` | Регистрация нового пользователя |
| `POST` | `/api/auth/login` | Авторизация и получение JWT |
| `GET` | `/api/leaderboard` | Получение сводной таблицы результатов (динамический расчет весов) |
| `GET` | `/api/teams` | Список зарегистрированных команд |
| `POST` | `/api/artifacts/repo` | Отправка Git-репозитория на автоанализ |
| `POST` | `/api/scores` | Сохранение экспертных оценок жюри в БД |

## 💻 Локальный запуск для разработки

### 1. База данных (PostgreSQL)

Установите PostgreSQL локально, создайте базу данных и скопируйте строку подключения в файл `.env` в корневой директории:

```env
DATABASE_URL=postgresql+asyncpg://USER:PASSWORD@localhost:PORT/DATABASENAME
```

### 2. Backend

Запуск из `DecisionHub/backend`:

```bash
uv run src/main.py
```

### 3. Frontend

Запуск из `DecisionHub/frontend`:

```bash
npm install
npm run dev
```
